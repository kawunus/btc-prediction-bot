# BTC Prediction Bot

Телеграм-бот для угадывания цены биткоина.

## Как работает

1. Админ в личку боту пишет `/start 20:00` — запускает глобальный раунд.
2. Участники в любом чате (где есть бот) или в личку пишут число — одна попытка.
3. Ровно в указанное время (МСК) бот берёт актуальную цену BTC.
4. Бот каждому участнику лично отправляет Топ‑5 победителей.

## Команды (админы)

| Команда | Описание |
|---------|----------|
| `/start HH:MM` | Запустить глобальный раунд на указанное время (МСК) |
| `/cancel` | Отменить текущий раунд, все ставки аннулируются |
| `/end` | Досрочно завершить раунд и определить победителя |
| `/status` | Посмотреть сколько ставок уже сделано |

## Ставки

- Участники просто пишут число (любой формат: `60000`, `60 000`, `60,000.50`, `60000,5`)
- Можно отправлять как в чат (если бот там есть), так и в личку боту
- Одна ставка на человека, изменить нельзя

## Запуск

```bash
cp .env.example .env
# Заполни .env своими значениями

docker compose up -d --build
```

## Настройка .env

- `BOT_TOKEN` — токен от @BotFather
- `ADMINS` — ID админов через запятую (первый — главный)
- Остальное — параметры базы данных

## Заметки

- Время раунда всегда в **МСК** (Europe/Moscow).
- Если бот перезапустится — активный раунд подхватится автоматически.
- Цена берётся с Binance (запасной источник — CoinGecko).
