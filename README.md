# BTC Prediction Bot

Телеграм-бот для угадывания цены биткоина.

## Как работает

1. Владелец пишет `/start 16:00` в беседу — открывается раунд.
2. Участники пишут число (цену BTC в USD) в чат или в ЛС боту — **одна попытка**.
3. Ровно в 16:00 UTC бот получает актуальную цену BTC.
4. Победитель — тот, чья ставка ближе всего к реальной цене.
5. Результаты публикуются в беседе.

## Команды

| Команда | Кто | Описание |
|---------|-----|----------|
| `/start HH:MM` | Владелец | Открыть раунд |
| `/cancel` | Владелец | Отменить текущий раунд |
| `/status` | Все | Посмотреть статус раунда |

## Запуск

```bash
cp .env.example .env
# Заполни .env своими значениями

docker compose up -d --build
```

## Настройка .env

- `BOT_TOKEN` — токен от @BotFather
- `ADMINS` — ID владельцев через запятую (первый — главный), можно один
- Остальное — параметры базы данных

## Заметки

- Время раунда всегда в **UTC**.
- Если бот перезапустится — активные раунды подхватятся автоматически.
- Цена берётся с Binance (запасной источник — CoinGecko).
